## How to build and execute?
### DNF Converter

- input: text file

- output: DNF

1. How to compile

     chmod 755 make 
            
        you must change mode for excuting ./make.
     
     ./make 'file name'
     
        ex) ./make test.txt

2. How to excute

    ./make 'file name'
        
        when you run ./make, the result is in solution file.
